# Forge 2 Qualifier – Aman

This repository contains the full Forge 2 qualifier submission.

## Project Structure

| Folder | Purpose |
|--------|---------|
| `openclaw/` | OpenClaw agent setup and configuration |
| `hermes/` | Hermes orchestrator agent |
| `kanban-api/` | Laravel backend (Kanban board API) |
| `kanban-ui/` | React frontend (Kanban board UI) |
| `slack-wiring/` | Slack integration and wiring configs |

## Tech Stack
- **Backend:** Laravel (PHP 8.2+) with SQLite
- **Frontend:** React
- **Agents:** OpenClaw + Hermes
- **Hosting:** Vercel/Netlify (frontend) + Render/Railway (backend)
